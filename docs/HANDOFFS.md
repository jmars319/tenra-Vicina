# Vicina by Tenra Handoffs

Vicina stays unique because it is a product-distinct hyperlocal coordination app with web/mobile-led participation.

## Produces

- Local board exports for operator review and development.
- Moderation/risk review packets when a signal, comment, report, or block needs deeper assessment.

## Consumes

- Sentinel risk briefs for suspicious contact, entity, or repeated-behavior review.
- Guardrail external action review before moderation actions become automated.
- Proxy-shaped public-facing copy only when operator-authored communication needs voice alignment.

Vicina should not absorb Sentinel, Guardrail, or Proxy. Those are reusable suite services that can support moderation and communication workflows.
